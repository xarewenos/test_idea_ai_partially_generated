Image Metadata and Content Comparison System

This project will extract metadata (such as EXIF data) from image files, compare filenames or metadata entries to a predefined list or a database, and identify if there is any textual similarity or error. The system will also allow sending email notifications when issues are detected, and the comparison of filenames or metadata can be based on edit distance.
Core Features:

Image Metadata Extraction:

  Extract EXIF metadata from image files (e.g., camera make, model, exposure time, location).

File Name and Metadata Comparison:

  Compare the metadata of an image (like the camera make and model) or filenames with a predefined list of known image details     using edit distance to measure similarity.

Email Notifications:

  Send email notifications if a certain condition is met, such as metadata inconsistencies or error messages during the process.

Error Handling:

  Handle potential errors during metadata extraction, file reading, and comparison processes with robust error handling using errno.
